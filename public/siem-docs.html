Overview

The SIEM API provides secure access to centralized logs from the AFL Underwriting Platform. Designed for integration with Splunk, QRadar, Azure Sentinel, and Elastic SIEM.

Base URL
https://afl.acc.ltd/api/siem
Log Types
  • security - Authentication & access events
  • sessions - User session lifecycle
  • access - HTTP request logs
  • application - App logs (debug/info/warn/error)
  • pii-audit - Sensitive data access tracking
Features
  • API Key + HMAC authentication
  • IP whitelist validation
  • Cursor-based pagination
  • CEF format support
  • Real-time & historical access

Authentication

All API requests (except /health) require three headers:

HeaderDescription
X-API-KeyYour SIEM API key
X-TimestampISO 8601 timestamp (within 5 min of server time)
X-SignatureHMAC-SHA256 signature
⚠️ IP Whitelist

Only requests from whitelisted IPs are accepted. Add your SIEM server IP to SIEM_ALLOWED_IPS environment variable.

Signature Generation

Generate HMAC-SHA256 signature with this payload:

// Payload = timestamp + method + path + queryString payload = "2025-02-06T10:30:00Z" + "GET" + "/api/siem/logs" + "?start=2025-02-06T00:00:00Z" signature = HMAC-SHA256(payload, api_secret)

Rate Limiting

  • Limit: 100 requests per minute
  • Response: HTTP 429 when exceeded
  • Header: retry_after indicates seconds to wait
GET /health

Health check. No authentication required.

{ "status": "healthy", "timestamp": "2025-02-06T10:30:00.123Z", "version": "7.4.2", "siem_configured": true, "storage": { "logs_today": 15234, "by_type": { "security": 523, "sessions": 1247 } } }
GET /logs

Query logs with filters and pagination.

Query Parameters
ParameterRequiredDescription
startYesStart time (ISO 8601)
endNoEnd time (default: now)
typeNoComma-separated: security,sessions,access,application,pii-audit
limitNoMax events (default: 1000, max: 10000)
cursorNoPagination cursor
{ "success": true, "meta": { "returned_count": 500, "has_more": true }, "logs": [ ... ], "cursor": "eyJ0cyI6..." }
GET /logs/correlation/:correlationId

Get all logs for a specific request/transaction.

GET /stats

Get log statistics for a date.

GET /schema

Get event schema for SIEM field mapping.

Code Examples

Node.js
const crypto = require('crypto'); const axios = require('axios'); async function fetchLogs(start, types) { const timestamp = new Date().toISOString(); const path = '/api/siem/logs'; const query = `?start=${start}&type=${types}`; const signature = crypto .createHmac('sha256', process.env.SIEM_API_SECRET) .update(timestamp + 'GET' + path + query) .digest('hex'); return axios.get('https://afl.acc.ltd' + path + query, { headers: { 'X-API-Key': process.env.SIEM_API_KEY, 'X-Timestamp': timestamp, 'X-Signature': signature } }); }
